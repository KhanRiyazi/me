Web Fundamentals (1 month)

Understand how the web works before diving into FastAPI.

Key Concepts:

HTTP Protocols

HTTP is the foundation of data communication on the web.

  • GET - Retrieve data
  • POST - Send data to create
  • PUT - Update existing data
  • DELETE - Remove data

Practice:

Think of a social media app. Which HTTP methods would be used for:

  • Loading your feed?
  • Creating a new post?
  • Editing a post?
  • Deleting a post?

REST API Principles

REST (Representational State Transfer) is an architectural style for designing networked applications.

  • Stateless - Each request contains all information needed
  • Client-Server - Separation of concerns
  • Cacheable - Responses can be cached
  • Uniform Interface - Consistent API design

FastAPI Core Concepts (2 months)

Now the exciting part! Let's dive into FastAPI.

Core Topics:

Creating Your First API Endpoint

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: str = None):
    return {"item_id": item_id, "q": q}

This creates two endpoints: a simple greeting and an item lookup.

Path & Query Parameters

Path parameters are part of the URL path, while query parameters come after the ? in the URL.

# Path parameter
@app.get("/users/{user_id}")
def get_user(user_id: int):
    # user_id is extracted from the URL
    return {"user_id": user_id}

# Query parameters
@app.get("/items/")
def get_items(skip: int = 0, limit: int = 10):
    # skip and limit are query parameters with default values
    return {"skip": skip, "limit": limit}

Advanced FastAPI (2 months)

Expand your skills with database integration and authentication.

Advanced Topics:

Database Integration

Connect FastAPI to databases like SQLite, PostgreSQL, or MongoDB.

# Example with SQLite
import sqlite3

def get_db_connection():
    conn = sqlite3.connect('database.db')
    conn.row_factory = sqlite3.Row
    return conn

@app.get("/users/{user_id}")
def get_user(user_id: int):
    conn = get_db_connection()
    user = conn.execute('SELECT * FROM users WHERE id = ?', (user_id,)).fetchone()
    conn.close()
    return dict(user) if user else None
